# NYC Rat Sightings Data Analysis  
**By Abdul Gadiru Sannoh**

## 📊 Project Overview
This project analyzes New York City rat sightings data to uncover patterns in urban pest activity and support data-driven decision-making in public health and infrastructure management.

## 🎯 Objectives
- Identify boroughs with the highest rat sightings over time  
- Analyze top location types with the most infestations  
- Visualize spatial distribution using geospatial mapping  

## 🛠️ Tools & Technologies
- R (tidyverse, dplyr, ggplot2, lubridate, sf)
- Data Cleaning & Transformation
- Geospatial Analysis
- Data Visualization

## 🔍 Methodology
The dataset was cleaned and structured by:
- Converting date fields into proper datetime format  
- Handling missing values and removing duplicates  
- Standardizing borough and location type fields  
- Filtering geographic coordinates for accurate mapping  

## 📈 Key Insights
- Brooklyn and Manhattan show the highest rat sighting concentrations  
- Residential and mixed-use buildings are major infestation hotspots  
- Seasonal and yearly trends influence rat activity  
- Geospatial mapping reveals high-density urban infestation zones
